SAE 104

  Objectifs

  1. Concevoir une base de données à partir des besoins d'un client.
  2. Implémentée, une interface web permettant d'interagir avec une base de données.

  Sujet

L'objectif de cette SAE était de réaliser une application web, répondant au sujet qui nous a été donné.
Dans notre cas, nous devions créer une application web, permettant a une association louant des vélos de gérer :

  • leurs vélos
  • les clients
  • les emprunts
De plus, cette application devait être conçu avec le framework flask.

  Livrables et Documents

Pour cette sae nous avions plusieurs livrables :

  • Rapport d'analyse des besoins
  • Inventaire des données et Modèle conceptuel de données
  • Le script de création base de données.
  • Jeu de données
  • Interfaces web (serveur en Python) pour afficher/lire/ajouter/modifier/supprimer des données dans des tables.
  • Requêtes avec des fonctions d’agrégation pour extraire des données et affichages du résultat dans le navigateur.
Vous pourrez trouver ces livrables dans ce répository

  Mon travail

Lors de ce travail, j'ai pris le rôle de chef de projet.
J'ai donc dû gérer l'équipe avec laquelle je travaillais, et j'ai donc réparti les tâches. Je me suis vite rendu compte que mes camarades de travail ne réalisaient pas leur part de travail et que je devrais donc faire leur par en plus de la mienne.
Pour qu'ils fassent leurs pars dans le projet. J'ai donc du mettre en place plusieurs mesure pour qu'ils fassent leur pars dans le projet :

  • Des créneaux où je leur proposais de leur expliquer comment le code fonctionnait et comment faire leurs parties.
    • J'ai notamment proposé un créneau d'une après-midi en semaine avant le rendu des parties personnelles ou j'ai expliqué aux membres qui sont venus comment fonctionnais leur partie et comment il pouvait l'implémenter.
  • Un system de pénalité ou ceux que ne rendait pas leur travail en temps et en heure risque de se faire radier du groupe.
Ce système a été mis en place après l'implémentation de l'interface web pour afficher/lire/ajouter/modifier/supprimer des données dans des tables.
J'ai donc dû implémenter cette partie qui était une note de groupe.

  Compétence Obtenue

Lors de ce travail, j'ai obtenu les compétences suivantes :
  1. Gestion d'une équipe qui ne travaille pas.
  2. Études des besoins d'une entreprise.
  3. Conception d'un produit répondant à des besoins.
  4. Développement d'une application web gérant une base de données.
