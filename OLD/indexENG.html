Objective of the project

  • Install and configure a machine.

Subject
The aim of this SAE was to configure a virtual machine according to specifications.
In addition, a manual had to be written, explaining how the machine was configured.

This workstation comprises a fixed machine, a portal computer and a docking station for the laptop.
On the right, you can see a screenshot of the virtual box.
Deliverables & Documents
For this sae we had several deliverables:
  • A file for studying and selecting solutions for the computer workstation, including:
    • A short presentation of the customer's request and how you intend to respond in the following sections
    • Description of proposed hardware architectures
    • Recommendation of a development workstation (recommended hardware architecture and software architecture), justifying the choice.

  • Virtual machine installation instructions, including an explanation of how to configure the virtual machine:

    • in text mode
    • desktop environment

  • A presentation where we had to present our virtual machine

During this job, I took on the role of project manager.
This meant I had to manage the team I was working with, so I divided up the tasks.
I soon realized that my workmates weren't doing their share of the work, so I'd have to do theirs as well as mine.
To get them to play their part in the project. So I had to implement several measures to get them involved in the project:

Summary
Soul Garden is a group of project which resolve around one of my project, a TTRPG Nepheshria.
Currently, it's composed of the two Core project, and a few subprojects that provide tools or add content to the main projects.
All the current and future project can be found inside the GitHub organization on the right.
Nepheshria Rule Book
The Nepheshria rule book is a book that contain the rules of the TTRPG.
The purpose of this book is to have the rules free from the lore influence.
So that anyone can use them even outside the Nepheshria universe.

A first draft of the book has been made, but as part of the Nepheshria universe was entangled in the rules, and after a few play test some of the rules were modified, so the book need a rewrite.
The first draft is available in the GitHub repository Nepheshria in the Soul Garden organization.
Please note that at this point, this repository is more a recoupment of my notes and unfinished work, than a structured project.

Nepheshria Lore Book
The Nepheshria lore book is a book that contain the lore of Nepheshria.
The different element that were previously in the rule book but were attached to the lore will be explained, but all game related info will be in a separated book. This includes the races and masteries.
The rule book will mainly talk about the main continent, the races that inhabit it, its history, and its important characters.
