SAE 106

  Objectifs

  1. Réaliser une étude économique d'une entreprise.
  2. Implémenter un site internet simple.

  Sujet

L'objectif de cette sae était de faire une étude économique sur une entreprise travaillant dans le domaine de l'informatique.
De plus, nous devions transcrire les différentes informations récoltées sous la forme d'un site web.
Le site web devait être hébergé sur github page, comme celui sur lequel vous vous trouvez.

À droite, vous pourrez trouver une capture d'écran de la page d'accueil du site web.
