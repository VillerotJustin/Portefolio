SAE 106

  Objectifs

  1. Réaliser une étude économique d'une entreprise.
  2. Implémenter un site internet simple.
  3. Transmettre des informations via un site web.

  Sujet

L'objectif de cette sae était de faire une étude économique sur une entreprise travaillant dans le domaine de l'informatique.
De plus, nous devions transcrire les différentes informations récoltées sous la forme d'un site web.
Le site web devait être hébergé sur GitHub page, comme celui sur lequel vous vous trouvez.

À droite, vous pourrez trouver une capture d'écran de la page d'accueil du site web.
Le lien du site web est disponible à la fin de cette page.

  Livrables et Documents

Vous pourrez trouver tout le code qui a été réalisé dans ce répository
Ainsi que les livrables de la sae 106, c’est-à-dire le site web et le rapport économique.

  Mon travail:

Lors de ce travail, j'ai pris le rôle d'un développeur.
Le chef de groupe, à après une conversation collective répartie les rôles.
J'ai donc dû m'occuper de la partie concernant la présentation de l'entreprise.
J'ai donc dû faire des recherches sur l'entreprise que nous avons sélectionnée Ankama.

  Compétence Obtenue

Lors de ce travail, j'ai obtenu les compétences suivantes :

  • Conception est l'implémentation d'un site web avec GitHub page.

  • Rechercher des informations sur un sujet.
