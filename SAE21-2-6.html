SAE 2.1-2-6

  Objectifs

  1. Implémentation d’un besoin client, en partant d’un besoin exprimé par le client.
  2. Développer une application Java, en fonction d'un cahier des charges précis avec des fonctions prédéfinie.
  3. Apprendre a utiliser un framework lors d'un projet informatique.
  4. Construire la structure d'un projet avec le paradigme MVC.
  5. Réaliser une comparaison algorithmique entre deux IA.
  6. Produire un code avec la batterie de tests associé.

  Sujet

Le sujet de cette sae était l'implémentation d'un jeu de plateaux a l'aide du framework Gamifire.
Nous devions choisir parmi cette liste de jeux de société :

  • Mastermind
  • Qwirkle
  • Poule-renard
  • kamisado
  • Backgammon
Notre groupe a donc choisi le Backgammon.
Vous pouvez voir une capture d'écran d'une partie à droite.

  Mon travail

Lors de ce travail, j'ai pris le rôle d'un membre du groupe.
Comme nous n'avions jamais travaillé avec le framework gamifire, les premières semaines, nous devions tester et comprendre son fonctionnement.
Pour ma part, je me suis chargé de comprendre comment fonctionnait les IA et de comment nous allions devoir les implémenter.
Bien que je n'aye pas implémenté les versions finales des IA, j'ai conçu les méthodes de base et les fondations de l'algorithme.
De plus, je me suis occupé du design de la partie graphique.
J'ai dessiné les différents pions, les dés ainsi que le plateau de jeux.

  Compétence Obtenue

Lors de ce travail, j'ai obtenu les compétences suivantes :
  1. Travailler au sein d'une équipe de développement.
  2. Concevoir une application en respectant le paradigme MVC.
  3. Utiliser un framework inconnue pour développer une application.
